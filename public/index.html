Hello! I am a web developer based in Seattle, Washington.

My largest project to date is a full-stack web site for Mercury Mail Services. Behind the login button on the public brochure site is a B2B customer management system which allows Mercury Mail's customers to enter and track data on their donors and donations. Customers can order routes based on geographic areas.

An extensive map component allows customers to map the neighborhoods where solicitation cards were mailed, as well as donations reported to customers' call centers by phone as well as by web. With mobile phone asset tracking, customers' trucks are shown on the map with a trail of where they have been.

This project is largely written in C# and ASP.NET, using a SQL Server database. The map components are written in JavaScript and jQuery.

More recently, I have built pixel-perfect mobile responsive internal websites for Boeing, and data-intensive internal consoles for Facebook using JavaScript, React, and PHP.
